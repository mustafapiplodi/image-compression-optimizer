# Image Compression Optimizer

A powerful, privacy-first image compression tool that reduces JPEG, PNG, and WebP file sizes by 60-90% directly in your browser. No uploads, unlimited usage, and completely free.

## Features

- **100% Private**: All processing happens in your browser - no server uploads
- **Unlimited**: No limits on file count, file size, or daily usage
- **Fast Compression**: Uses advanced algorithms with Web Workers for optimal performance
- **Batch Processing**: Compress multiple images simultaneously
- **Before/After Comparison**: Interactive slider to compare original vs compressed
- **Format Conversion**: Convert images to WebP for additional size savings
- **Dark Mode**: Comfortable viewing in any lighting condition
- **PWA Support**: Install as an app and use offline
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile

## Tech Stack

- **React** + **TypeScript** - Type-safe component architecture
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - Beautiful, accessible components
- **browser-image-compression** - High-quality image compression
- **JSZip** - Batch download as ZIP

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Usage

1. **Upload Images**: Drag and drop or click to select images
2. **Adjust Settings**: Configure quality (recommended: 70-85%)
3. **Advanced Options**:
   - Convert to WebP format
   - Set max dimensions for resizing
4. **Compress**: Images are automatically compressed
5. **Compare**: Use the comparison slider to see before/after
6. **Download**: Download individually or all as a ZIP file
